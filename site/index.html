Track Your Claude.ai Usage

Keep track of your Claude.ai plan usage at a glance. A lightweight macOS menu bar app for power users.

Download for macOS View on GitHub
Requires macOS 14.0 Sonoma or later

Everything you need to stay informed

Powerful features designed for Claude.ai power users who want complete visibility into their usage.

Real-time Monitoring

Track your 5-hour session limits, 7-day weekly usage, and Sonnet-specific limits in real-time. Never get caught off guard by hitting your limits.

Menu Bar Indicator

Color-coded gauge icon in your menu bar. Green for safe, yellow for warning, red for critical - see your status at a glance.

Smart Notifications

Configurable warning and critical thresholds. Get notified before you hit your limits, not after.

Auto-refresh

Automatic usage updates every 1, 5, or 10 minutes. Set it and forget it - your usage data is always up to date.

JSON Export

Export usage data as JSON for integration with external tools, scripts, and dashboards (e.g. Claude Code statusline).

Secure Storage

Session keys are encrypted and stored securely in the macOS Keychain. Your credentials never leave your machine.

Install in seconds

Get up and running with a single command using Homebrew.

Terminal
$ brew install eddmann/tap/claudemeter
